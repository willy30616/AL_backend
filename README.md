## 執行專案

$ java -jar com.aluminum.second-0.0.1-SNAPSHOT.war

### 開發環境

eclipse專案 Meaven打包 spring boot框架 SQL SERVER資料庫

確認java環境 java -version: 1.8.0_301

確認資料庫設定 application.properties
